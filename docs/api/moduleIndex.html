Module Index

  • pythagora - Pythagora is an agent-based modeling package that allows for the construction of experiments to test the impact of select types of social networks on production and consumption outcomes. Specifically, _Pythagora_ tests how individuals (agents) in small-world social networks and scale-free social networks (as well as control versions of these networks) buy and sell new types of pottery based on interactions within their social networks.
